## ğŸ’» Usage

### Customization

To personalize the portfolio for your own use:

1. **Update Personal Information**: 
   - Replace `[Your Name]` placeholders in `index.html`
   - Update the meta description and title tags

2. **Add Your Projects**:
   - Edit the project articles in the `#projects` section
   - Replace placeholder images (`project1.png`, `project2.png`)
   - Add appropriate `data-category` attributes for filtering

3. **Modify Skills**:
   - Update the skills list in the `#skills` section with your own technologies

4. **Configure Contact Form**:
   - Update the form `action` attribute to point to your backend endpoint
